# to-do-list-ts

A dynamic To-Do List web application built with TypeScript, allowing users to add, remove, and clear tasks.  
The app supports multiple languages using i18n and demonstrates structured, object-oriented frontend development with a clean template system.

This project was created to practice TypeScript, modular design, DOM manipulation, and internationalization.
